Resummino Quick Start

1. Prerequisites

You will need some Boost headers, the GNU Scientific Library, the LHAPDF library and for Resummino-2.0.0 also the LoopTools library. The first two can be installed in RHEL/Scientific Linux or Fedora by using:

$ sudo yum install boost boost-devel gsl gsl-devel

In Debian/Ubuntu you can use a similar command for the same purpose:

$ sudo apt-get install libboost-dev libgsl0-dev

You will need to install the LHAPDF library by following the instructions at the website, if you have not done so yet. You should also write down the path where LHAPDF is installed to.

For the compilation and installation of Resummino 2.0.0 follow the included Readme file.

2. Compilation and installation

You can issue the following commands in a terminal to download, compile and install Resummino:

$ wget {{ site.tarball_url }}resummino-{{ site.version }}.tar.bz2
$ tar -xvf resummino-{{ site.version }}.tar.bz2
$ cd resummino-{{ site.version }}
$ mkdir build
$ cd build
$ cmake ../ -DLHAPDF=/usr/local/lhapdf -DCMAKE_INSTALL_PREFIX=/usr/local/resummino
$ make
$ make install

In this example, LHAPDF should be installed at /usr/local/lhapdf and Resummino will be installed to /usr/local/resummino. Of course you should specify your own paths. If you do not specify the paths, default paths will be tried. You can also use curl -O instead of wget. Note that Resummino currently does not work with the newest gsl version. If you want to use Resummino and observe any problem regarding gsl you have to download an older version and link it via: 

$cmake ../ -DCMAKE_PREFIX_PATH=path/to/libgsl
Also note that you do not have to specify where resummino hsa to be installed. If you just do make all necessary binary files should be contained in the build folder.

3. Usage

The following command will run the included example:

$ resummino input/resummino.in

You can also specify the --lo or --nlo options to compute only up to LO or NLO.

You can see the comments in input/resummino.in for more details on the input file format.
